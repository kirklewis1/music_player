Python Music Player
A simple music player built in Python using Tkinter library.

Requirements
Python 3.x
Tkinter
Pygame
How to run
Clone the repository.
bash

git clone https://github.com/[username]/python-music-player.git
Install Tkinter and Pygame.

pip install tkinter
pip install pygame
Run the player.

python music_player.py
How to use
Add songs to the music folder.
Select a song from the list and press the play button.
Use the previous, play/pause, and next buttons to control the playback.

Features
Add and play songs
Play, pause, and skip controls
Display currently playing song
